Privacy Policy

Information Collection

This application collects minimal anonymous usage statistics to help us understand how the service is being used and improve it. We do not collect any personally identifiable information. The anonymous data we collect includes:

  • Number of visitors
  • Page views
  • Geographic region (country level only)
  • Device type (mobile/desktop)
  • Session duration

We do not use cookies or any other tracking mechanisms that could identify individual users.

Data Processing

All calculations are performed using artificial intelligence to analyze current news events. The results are speculative and should not be considered as definitive predictions. We use OpenAI's GPT model through OpenRouter for text analysis and NewsAPI for gathering news data.

Data Storage

We store the following types of data:

  • Historical data of our calculations and analyzed news summaries
  • Aggregated anonymous usage statistics
  • System performance metrics

All stored data is anonymous and is used solely for improving the service and displaying trends. We do not store any personal information or individual user data.

Analytics

We use basic analytics to understand how our service is being used. This helps us:

  • Monitor system performance
  • Identify and fix issues
  • Improve user experience
  • Make informed decisions about future updates

All analytics data is collected anonymously and aggregated. Individual users cannot be identified from this data.

Third-Party Services

This application uses the following third-party services:

  • NewsAPI - For gathering current news data
  • OpenRouter - For AI-powered analysis
  • Chart.js - For data visualization
  • Simple Analytics - For anonymous usage statistics

Data Retention

Anonymous usage statistics are retained for up to 12 months. Historical probability calculations and news summaries are retained indefinitely for trend analysis and historical context.

Your Rights

Since we don't collect any personal information, there is no personal data to access, modify, or delete. The anonymous statistics we collect cannot be linked to individual users.

Disclaimer

The probability calculations provided by this application are speculative and based on AI interpretation of current news events. They should not be taken as definitive predictions or cause for panic. This is an educational and entertainment tool, not a scientific instrument.

Updates to Privacy Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.

Contact

For any questions regarding this privacy policy or the application in general, please visit our GitHub repository.
